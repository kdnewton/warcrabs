Figuring Crab Stats:

Optimum Shell Capacity:
Same as the crab’s Strength.
Maximum Shell Capacity:
The crabs Dexterity and Strength combined.
Movement:
Same as Dexterity except when Weight is greater than Strength. In that case it is (Dexterity minus (Weight minus Strength)).
